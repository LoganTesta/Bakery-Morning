== Description ==
Use this theme for your bakery website or for any other business.  Customize your site including meta description and keywords, location, 
store hours, phone, social links, and more.  Enjoy.

== Installation ==
1. Upload the theme in your directory /wordpress/wp-content/themes/.
2. In WordPress admin, go to Appearance > Themes.  Find Bakery Morning, and click 'Activate' to activate it. 
3. Customize in Appearance > Customize.
